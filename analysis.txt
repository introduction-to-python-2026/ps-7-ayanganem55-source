
# Data Analysis Report: Palmer Penguins Dataset

## Dataset Chosen
The **Palmer Penguins** dataset was chosen for this analysis. It is a popular, clean dataset available within the `seaborn` library that provides measurements for three different species of penguins found in the Palmer Archipelago, Antarctica.

## Features Analyzed
The analysis focused on the following numerical features:
*   `bill_length_mm`
*   `bill_depth_mm`
*   `flipper_length_mm`
*   `body_mass_g`

## Observed Correlation
The primary correlation scatter plot generated (`correlation_plot.png`) visualizes the relationship between `flipper_length_mm` and `body_mass_g`.

### Key Observations:
1.  **Strong Positive Correlation:** There is a clear and strong positive linear correlation between a penguin's flipper length and its body mass. As the flipper length increases, the body mass consistently increases.
2.  **Visual Confirmation:** The red regression line in the saved figure effectively captures this upward trend, confirming the relationship.
3.  **Species Clustering (Implicit):** While the main correlation plot averages the view across all species, the individual scatter plots generated by the script show that different species occupy distinct clusters (e.g., Gentoo penguins are generally larger with longer flippers than Ad√©lie or Chinstrap penguins), which drives this overall positive correlation.

In summary, the data suggests that longer flippers are a very good predictor of higher body mass in these penguin species.
